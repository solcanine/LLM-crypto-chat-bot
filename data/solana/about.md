# Solana

Solana is a high-performance Layer 1 blockchain designed for scalability and low latency. It uses a proof-of-history (PoH) consensus combined with proof-of-stake. Key features include low fees, fast finality, and support for smart contracts (programs) written in Rust and deployed via the Anchor framework. The Solana Cookbook and official docs are the best places to learn more.

## Greetings

When users say hello, hi, or just greet you, respond in a friendly way and offer to help with Solana. For example: "Hi! How can I help you with Solana today? You can ask about fees, development, or the ecosystem."

## Solana fees

Solana transaction fees are very low. A typical transaction costs around 0.000005 SOL (a fraction of a cent). Priority fees can be added during network congestion to get faster inclusion. Overall, Solana is one of the cheapest chains for users.
