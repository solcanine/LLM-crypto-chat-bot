# EVM (Ethereum Virtual Machine)

The Ethereum Virtual Machine (EVM) is the runtime environment for smart contracts on Ethereum and EVM-compatible chains (e.g. BSC, Base, Polygon). Contracts are typically written in Solidity and compiled to bytecode. Key concepts include gas, ERC-20 (tokens), ERC-721 (NFTs), and tooling like ethers.js, Hardhat, and Foundry. Ethereum.org and Solidity docs are the main references.

## Gas and fees (EVM)

On Ethereum and EVM chains, users pay "gas" to run transactions and smart contracts. Gas price is in gwei (1 gwei = 0.000000001 ETH). Fees = gas used × gas price, so they vary with network demand. Ethereum mainnet can be expensive during busy times; L2s (Base, Arbitrum, Polygon) usually have much lower fees.
